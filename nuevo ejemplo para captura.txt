Cómo validar rápido
Arranca la API y la web.
Captura en el orden de arriba o ejecuta python scripts\seed_demo_veracruz_villahermosa.py y compara pantalla por pantalla con esta lista.
Abre scripts\factura_demo_veracruz_villahermosa.txt tras el seed y compáralo con la factura en el sistema.
Si en algún paso el sistema muestra otro folio (COT/OS/FAC), es normal: solo asegúrate de enlazar cotización → flete → orden → despacho → factura con los mismos códigos (FLT-DEMO-VVER-VHS-001, etc.).
